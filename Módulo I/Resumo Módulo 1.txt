No Módulo I vimos sobre o pensamento computacional sendo este um pensamento sistemático e eficiente tanto na formulação quanto na resolução de problemas, podendo ser utilizado em diferentes ramos do conhecimento, sendo este, baseado em 4 pilares: Decomposição, Reconhecimento de Padrões, Abstração e Algoritmos.
Entretanto, temos habilidades complementares que precisam ser definidas, sendo este o raciocínio lógico e aperfeiçoamento. Neste modelo temos três classificações indução, dedução e abdução.
Introduzindo a lógica de programação o foco é apresentar o conceito de lógica aplicando a programação como um processo de pensamento atrelado ao conceito de algoritmos e resolução de problemas.
A lógica seria a organização e planejamento das instruções assertivas em um algoritmo a fim de viabilizar a implantação de um programa.
Na linguagem de programação apresenta parâmetros de programação existentes, baseadas em um ou mais paradigmas que é a forma de resolução de problemas com diretrizes e limitações especificas de cada paradigma utilizando linguagem de programação.
Assim iniciamos no git que nada mais é do que um sistema de versionamento de código distribuído utilizado para projetos dedesenvolvimento de software, quer dizer que trabalha como um intermediador entre um repositório local e um remoto. Atualmente se tem o GitHub no mercado que domina como plataforma de distribuição e colaboração de códigos.
O Git abrange 3 camadas: o chamado “Working Directory”, que é onde você está codificando o projeto, “Stage”, que é uma camada onde você adiciona suas modificações de forma não permanente, ou seja, nesta camada você pode resetar as alterações que você incluiu no Stage ou então partir para a próxima camada, a chamada de “HEAD”, quando você confirma suas alterações fazendo um “commit”. Porém, para que o código chegue até o repositório remoto é necessário um comando muito simples chamado git push, sendo todo esse fluxo manipulado através do Git Bash nada mais é do que uma aplicação que emula um terminal e permite que desfrute dos comandos Git.
